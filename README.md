# About

The Financial Performance Dashboard provides a clear view of a company‚Äôs overall financial health. It tracks key metrics such as total sales, total profit, total units sold, and profit margin. The dashboard includes visual charts showing sales trends by year, monthly profit performance, and profit distribution across different countries. It also highlights which products and discount bands contribute most to sales and profits. With this dashboard, users can easily identify top-performing regions and products, monitor changes in financial results over time, and make informed business decisions. The filters for country, year, segment, and product allow users to analyze data from different perspectives and gain deeper insights into company performance.
